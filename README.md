
AI-powered Virtual Drawing Application 

🚀 About the Project

AirSketch AI is a computer vision project that enables users to draw in the air using hand gestures without touching a mouse or screen.

Using MediaPipe's real-time hand tracking and OpenCV, the application detects finger movements and converts them into digital drawings.

---

✨ Features

- 🎨 Air Drawing
- ✋ Hand Gesture Recognition
- 🖌 Multiple Brush Colors
- 🧽 Eraser
- 💾 Save Drawing
- 🗑 Clear Canvas
- ⚡ Real-Time Performance
- 📷 Live Webcam Tracking

---

🛠 Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
